# Premium Habit Tracker 🔥

A modern, highly-polished, offline-first web application designed to track the intricacies of your daily habits. It operates completely serverless, relying strictly on your browser's Local Storage for maximum privacy and explosive speeds.

## Features

- **Gamified Streaks**: Keep your boxes ticked consecutively and watch dynamic flame icons ignite in your column headers!
- **Lifetime Analytics Heatmap**: Every single action you take is charted across a beautiful, endlessly-scaling GitHub-style colored matrix in the Analytics dashboard.
- **Drag-and-Drop Organization**: Click and drag your habit headers horizontally to flawlessly rebuild your tracker layout natively without refreshing.
- **100% Private Offline Storage**: No cloud accounts. Zero tracking. Your data is isolated exclusively to your local machine.
- **Instantly Installable (PWA)**: Using modern Service Workers, Mobile and Desktop users can click "Add to Home Screen" to install it as a fully un-tabbed, native fullscreen application!
- **Instant JSON Exporting**: Fearful of clearing your browser cache? Click the Backup button to instantly download your entire history into a secure JSON text file.

## Getting Started

Because everything is calculated flawlessly in the browser, deployment is instantaneous. 

1. Simply clone this repository.
2. Open `index.html` directly in any web browser. 
3. Click on the default "Ignite the Journey" habit to change its color, or rename it.

Or, visit the live deployment hosted directly on GitHub Pages!

## Technologies

- Standard Vanilla HTML5
- Advanced CSS3 Grid + Subgrid architecture with custom Glassmorphism designs.
- Pure ES6 JavaScript. No build tools (Webpack/Vite) required.
