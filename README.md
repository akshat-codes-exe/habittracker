A modern, highly-polished, offline-first web application designed to track the intricacies of your daily habits. It operates completely serverless, relying strictly on your browser's Local Storage for maximum privacy and explosive speeds.

---

## 🚀 Key Features

* **Gamified Streaks:** Keep your boxes ticked consecutively and watch dynamic flame icons ignite in your column headers!
* **Lifetime Analytics Heatmap:** Every action is charted across a beautiful, GitHub-style colored matrix and visual pie charts.
* **Drag-and-Drop Organization:** Click and drag your habit headers horizontally to rebuild your layout natively without refreshing.
* **Integrated To-Do List:** A simple, built-in task manager for immediate "To-Dos" alongside your long-term habits.
* **100% Private Offline Storage:** No cloud accounts. Zero tracking. Your data is isolated exclusively to your local machine.
* **Data Portability:** Export your entire history to **Excel, PDF, or JSON** anytime.
* **Instantly Installable (PWA):** Use "Add to Home Screen" to install it as a native, fullscreen application on desktop or mobile.

---

## 🛠️ Technologies Used

* **HTML5:** Semantic structure for a modern web experience.
* **CSS3:** Advanced Grid and Subgrid architecture with custom Glassmorphism UI.
* **Vanilla JavaScript:** Pure ES6 logic for performance—no bulky frameworks or build tools required.
* **AI-Assisted Development:** Built using modern tools like Cursor and Antigravity.

## 🏁 Getting Started

Because everything is calculated in the browser, deployment is instantaneous.

1.  **Clone this repository:**
2.  **Run the App:** Open `index.html` directly in any modern web browser.
3.  **Customize:** Click on the default "Ignite the Journey" habit to change its color or rename it.

**Live Demo:** [Visit the Live Deployment](https://akshat-codes-exe.github.io/habittracker/)
